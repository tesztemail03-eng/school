Adatelemzési esettanulmány – Projektterv
Projekt célja
Egy valós adatelemzési probléma megoldása Python és BI eszközök segítségével.
A cél egy olyan elemzés készítése, amely bemutatja:

az adattisztítási folyamatot

az alkalmazott statisztikai módszereket

a modellépítést és diagnosztikát

az eredmények üzleti értelmezését

Tervezett tartalom
1) Adatforrások
nyilvános dataset (pl. Kaggle) vagy saját adat

adattisztítás: hiányzó értékek, duplikátumok, outlierek kezelése

2) Elemzési módszerek
leíró statisztikák

korrelációs elemzés

multikollinearitás vizsgálat (VIF)

lineáris regresszió (statsmodels)

modell diagnosztika

3) Vizualizációk
Python (Matplotlib, Seaborn)

Power BI vagy Tableau dashboard

4) Eredmények
fő összefüggések bemutatása

modell teljesítménye

üzleti következtetések

5) Javaslatok
ajánlások a „megrendelőnek”

továbbfejlesztési lehetőségek





A beadandót a félév végére kell elkészíteni, és valószínűleg a héten készen leszek vele. A projekthez a Spotify legnépszerűbb dalait elemzem.


Mit fogok elemezni a Spotify adatelemzésben?
1) A dalok népszerűsége és stream száma
   
Mely dalok a legnépszerűbbek?

Milyen tényezők befolyásolják a stream számot?

Van‑e összefüggés a megjelenési év és a népszerűség között?


2) Előadók és műfajok elemzése
Kik a legtöbbet streamelt előadók?

Mely műfajok teljesítenek a legjobban?

Milyen különbségek vannak műfajok között hangulati jellemzőkben?


3) Hangulati és akusztikai jellemzők vizsgálata
A Spotify audio‑feature‑ök alapján:

danceability

energy

valence (pozitív hangulat)

tempo

acousticness

instrumentalness

Elemzem, hogy ezek hogyan kapcsolódnak a dalok sikerességéhez.


4) Korrelációk és összefüggések
Mely változók függnek össze erősen egymással?

Mely tényezők mutatnak kapcsolatot a stream számmal?

Multikollinearitás vizsgálata (VIF)


5) Prediktív modell építése
Regressziós modell a stream szám előrejelzésére

Modell teljesítményének értékelése (R², RMSE)

Fontosabb változók azonosítása (feature importance)


6) Vizualizációk és dashboard
Top előadók és műfajok

Hangulati jellemzők eloszlása

Stream trendek évek szerint

Interaktív BI dashboard (Power BI / Tableau)


7) Üzleti következtetések
Milyen jellemzők növelik egy dal sikerességét?

Milyen mintázatok alapján lehet új dalokat pozicionálni?

Milyen ajánlásokat lehet adni kiadóknak vagy előadóknak?
