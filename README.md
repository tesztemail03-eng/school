# school

Adatelemzési esettanulmány – Projektterv
Projekt célja
Egy valós adatelemzési probléma megoldása Python és BI eszközök segítségével.
A cél egy olyan elemzés készítése, amely bemutatja:

az adattisztítási folyamatot

az alkalmazott statisztikai módszereket

a modellépítést és diagnosztikát

az eredmények üzleti értelmezését

Tervezett tartalom
1) Adatforrások
nyilvános dataset (pl. Kaggle) vagy saját adat

adattisztítás: hiányzó értékek, duplikátumok, outlierek kezelése

2) Elemzési módszerek
leíró statisztikák

korrelációs elemzés

multikollinearitás vizsgálat (VIF)

lineáris regresszió (statsmodels)

modell diagnosztika

3) Vizualizációk
Python (Matplotlib, Seaborn)

Power BI vagy Tableau dashboard

4) Eredmények
fő összefüggések bemutatása

modell teljesítménye

üzleti következtetések

5) Javaslatok
ajánlások a „megrendelőnek”

továbbfejlesztési lehetőségek
